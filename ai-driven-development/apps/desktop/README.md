# Desktop App

Tauri v2 + React desktop application for AI DJ Assist.

## Features

- **Dark Theme UI**: High-contrast mode for club/stage environments
- **Analytics Dashboard**: Energy graph, genre donut, mixing timeline (custom SVG)
- **Camelot Wheel**: Interactive harmonic mixing visualization with click-to-filter
- **6-Step Onboarding**: Library import, preferences, account setup
- **Streaming Integration**: Beatport, Tidal, Spotify search with ISRC dedup
- **Cross-Platform**: macOS and Windows (MSI/NSIS)
