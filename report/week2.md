# 1월 2주차 활동 보고서

## 1. 주요 활동: 기술 검증 및 페지빌리티 테스트
- **음악 데이터 수집**: 테스트용 Drill Music 음원 데이터를 수집하고 오디오 신호 특성 분석.
- **임베딩 모델 리서치**: 음악 간 유사도 측정을 위한 CLAP 모델의 적합성 검토.
- **프로토타입 스크립트 작성**: 단순 오디오 로드 및 기본 특징(BPM, 에너지) 추출 기능 테스트.

## 2. 기술적 세부사항
- **특징 추출**: `librosa`를 활용하여 음원에서의 Tempo 및 Root Mean Square (RMS) 에너지 계산 로직 검증.
- **데이터 구조화**: 분석된 메타데이터를 효율적으로 관리하기 위한 JSON 스키마 설계.

## 3. AI 활용 경험 (Cursor AI: Composer-1)
- **활용 사례**: 다양한 오픈소스 라이브러리의 예제 코드를 바탕으로 한 테스트용 스크립트 자동 생성.
- **성과**: 기술 검증 단계에서 반복적인 코드 작성을 줄이고, 여러 라이브러리 간의 호환성을 빠르게 확인 가능했음.

## 4. 성과 요약
- 핵심 AI 로직 구현을 위한 기술적 타당성 확인 완료.
- 음악 분석을 위한 기초 알고리즘 프로토타입 확보.
