# PNU AI Booster - DrillMusic (AI DJ Project)

**AI Native 개발 방식을 통한 차세대 AI DJ 시스템 구현 프로젝트**

이 프로젝트는 **PNU AI Booster** 프로그램의 일환으로 진행되며, AI Tool을 적극 활용하여 전문 DJ의 워크플로우를 혁신하는 실용적인 솔루션을 개발합니다.

## 📖 프로젝트 개요 (Overview)

DrillMusic은 AI, App, Cloud 세 분야의 개발자가 협업하여 **AI DJ 시스템**을 구축하는 프로젝트입니다. 단순한 소프트웨어 개발을 넘어, 최신 AI 도구를 활용한 **AI Native 개발 방법론**을 실제 프로젝트에 적용하고 그 효용성을 검증하는 데 중점을 둡니다.

## 🎯 프로젝트 목적 (Objective)

본 프로젝트의 핵심 목표는 **AI Native 개발의 현 주소를 파악**하고, 미래 개발자에게 필요한 소양을 정의하는 것입니다. 이를 위해 다음과 같은 접근 방식을 취합니다.

*   **개발 방식의 비교 분석**: 인간 주도 개발(Human-Driven)과 AI 주도 개발(AI-Driven) 과정을 병행하며 두 방식의 장단점과 특징을 면밀히 비교합니다.
*   **AI 주도 개발의 가능성 탐구**: 프로젝트의 단순 완성도보다는 **AI 주도 개발 결과물의 퀄리티 분석**과 **AI 협업 프로세스 탐구**에 집중합니다.
*   **미래 역량 정의**: AI 시대에 개발자가 갖춰야 할 필수적인 역량과 소양을 도출합니다.

## 📂 프로젝트 구조 (Project Structure)

이 저장소는 개발 방식에 따라 크게 두 가지 메인 디렉토리로 구성되어 있습니다. 각 디렉토리 내부에는 해당 방식에 따른 상세 프로젝트 내용이 포함되어 있습니다.

```bash
.
├── 🤖 ai-driven-development/    # AI 에이전트 및 도구가 주도하여 개발한 프로젝트 영역
├── 👨‍💻 human-driven-development/ # 인간 개발자가 주도하여 개발한 프로젝트 영역
├── 📝 report/                   # 프로젝트 진행 보고서 및 분석 자료
└── 📄 README.md                 # 프로젝트 메인 문서
```

*   **ai-driven-development/**: AI의 코딩 및 설계 능력을 중심으로 진행된 결과물입니다.
*   **human-driven-development/**: 전통적인 방식 혹은 인간 개발자의 판단을 중심으로 진행된 결과물입니다.

> 💡 **참고**: 각 프로젝트의 실행 방법 및 상세 기술 스택은 해당 디렉토리 내의 `README.md`를 참고해 주세요.
